# VDL-Rust and VDL-C Artefacts for IEEE DCOSS-IoT

This repository corresponds to the code analyzed in the article:

B. Thapa, D. Alfonso, L. Bini, L. Mapelli, K. Schleiser, R. Fouquet, and E. Baccelli, "[Embedded Rust or C Firmware? Lessons from an Industrial Microcontroller Use Case with Ariel OS](https://arxiv.org/abs/2604.25679)", in Proceeedings of IEEE DCOSS-IoT Workshop on on IoT Applications and Industry 5.0 (IoTI5), to appear June 2026.

We are in the process of moving the open source code for VDL-C and VDL-Rust respectively to separate repos under the STMicroelectronics org at https://github.com/stm32-hotspot/ 

We'll link to these repos asap here. Stay tuned!

In the meantime, try Ariel OS out on the hardware:

- Open source Ariel OS support for the STEVAL-MKBOXPRO available at https://ariel-os.github.io/ariel-os/dev/docs/book/boards/st-steval-mkboxpro.html 

- First steps guide: https://github.com/ariel-os/ariel-os#getting-started 

- Open source Rust drivers for ST MEMS available at https://github.com/STMicroelectronics/st-mems-rust-drivers
